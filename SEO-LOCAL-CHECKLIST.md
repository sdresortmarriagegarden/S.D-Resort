## Before Going Live

- Replace the disabled footer Facebook icon with the real Facebook page URL when available.
- Upload the site to HTTPS hosting.
- Confirm `/robots.txt` and `/sitemap.xml` load publicly.

## Google Setup

- Create or claim the Google Business Profile.
- Primary category: Wedding venue.
- Add secondary categories only if accurate: Event venue, Banquet hall.
- Add services: Wedding Ceremony, Reception, Engagement, Sangeet, Mehendi, Haldi Ceremony, Guest Rooms, Catering Support.
- Upload photos: front gate, wedding stage, garden seating, room, lawn, night decor, entrance.
- Add final website URL, phone number, opening hours, address, and Instagram/Facebook links.
- Set up Google Search Console and submit `sitemap.xml`.
- Set up GA4 only after the final domain is live.

## Ongoing Ranking Work

- Ask real customers for Google reviews after each event.
- Reply to every review with natural mentions of the wedding venue, marriage garden, Sikar, and the event type.
- Create consistent listings on Justdial, WeddingWire India, WedMeGood, Sulekha, Instagram, Facebook, and relevant Sikar/Rajasthan directories.
- Add new real venue photos regularly to the website and Google Business Profile.
